# Security Policy

## Reporting a Vulnerability

If you find a security issue (for example, unsafe handling of API keys or potential fund loss due to a bug), please:

- Do **not** open a public GitHub issue with sensitive details.
- Email me at: artemborlandex@gmail.com
- Include reproduction steps and environment details.

I will acknowledge the report within 72 hours and aim to provide a fix or mitigation plan as soon as possible.
